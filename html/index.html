Introduction

In last few years web technologies became a silver bullet for UI definition and crossplatform development. However we still have a lot of discussion on it pros and cons. The most critical issue developers face when creating web apps is performance comparing to native applications. There are a lot of tricks and knowledge you need to know in order to produce HTML / JS application which is as fast as iOS or Android native. This e-book is a short practical guide into web performance which cover all aspects you need to create fast applications and stunning interactive experiences able to challenge native platforms.
